
POKEMON CARD SCANNER PRO

Works on:
- iPhone
- Android

FEATURES
- camera scanning
- OCR text recognition
- PokemonTCG API lookup
- CSV export for Cardmarket

INSTALL

1 unzip files
2 upload to GitHub Pages
3 open website on phone
4 add to home screen

SCAN TIPS

- place cards in grid
- no overlap
- good lighting
- white background

This template supports up to ~50 cards per photo with future AI detection.
